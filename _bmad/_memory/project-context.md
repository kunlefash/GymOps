# GymOps - Project Context

## Project Overview
GymOps is a gym operations management platform designed to streamline gym management, member tracking, and operational workflows.

## Tech Stack
- **Frontend**: Next.js 15, TypeScript (strict), Tailwind CSS
- **Backend**: Node.js with Express/Fastify, TypeScript
- **Database**: PostgreSQL with Prisma ORM
- **Deployment**: Vercel (frontend + serverless API)
- **Testing**: Jest (unit), React Testing Library (components), Playwright (E2E)

## Architecture Principles
- Clean architecture with clear separation of concerns
- API-first design
- Mobile-responsive (gym managers on the go)
- TDD mandatory - tests written before implementation
- Conventional commits enforced

## Current State
- Landing page with waitlist signup (Formspree integration)
- Deployed on Vercel
- No backend or database yet
- BMAD framework initialized for structured development
