# Create Epics & Stories

Decompose PRD into epics and implementable stories.

## Activation
1. Read `_bmad/bmm/agents/pm.md` for PM persona
2. Read `_bmad/bmm/config.yaml`
3. Load PRD and Architecture docs from `_bmad-output/planning-artifacts/`
4. Execute: `_bmad/bmm/workflows/3-solutioning/create-epics/steps/step-01-decompose.md`
5. Use templates from `_bmad/bmm/data/templates/epic-template.md` and `story-template.md`
6. Epics saved to `_bmad-output/planning-artifacts/`
7. Stories saved to `_bmad-output/implementation-artifacts/stories/`
