# Sprint Planning

Plan a new sprint from the backlog.

## Activation
1. Read `_bmad/bmm/config.yaml`
2. Load all stories from `_bmad-output/implementation-artifacts/stories/`
3. Execute: `_bmad/bmm/workflows/4-implementation/sprint-ops/steps/step-01-plan.md`
4. Create sprint file in `_bmad-output/implementation-artifacts/sprints/`
